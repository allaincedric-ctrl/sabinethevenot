# Site de présentation personnelle

Site statique en français pour présenter Sabine Allain Thevenot, disponible pour de nouvelles opportunités.

## Fichiers

- `index.html` — contenu du landing page
- `styles.css` — mise en forme responsive
- `script.js` — interaction du bouton de contact
- `photo.svg` — image de présentation placeholder

## Exécution locale

Ouvrez `index.html` dans votre navigateur ou servez le dossier via un serveur local.

## Remarques

Remplacez `photo.svg` par une photo réelle si vous le souhaitez, et ajustez le texte dans `index.html` pour refléter le CV réel.
